README - Dark UI Kit
Modern Dark-Themed UI Asset Pack

Thank you very much for downloading the Dark UI Kit.
This package was designed to help you quickly build clean, modern, and visually polished user interfaces for your game projects.
All assets are provided as PNG files, and the folder structure is organized to make your workflow simple and comfortable, no matter which engine you use.

====
Folder Structure

DarkUI_KIT/
├─ UI_Elements/
│ ├─ Buttons/
│ ├─ Panels/
│ ├─ ListItems/
│ ├─ ProgressBars/
│ ├─ Sliders/
│ ├─ Scrollbars/
│ ├─ Icons/
│ ├─ Controllers/
│ ├─ Decorative/
│ └─ CircleMenu/
├─ Backgrounds/
├─ VisualAssets/
└─ Preview/

The UI_Elements folder is divided by component type so you can easily locate the exact element you need while building menus, HUDs, or in-game widgets.

====
Included Assets

Here is a brief overview of what is included in the kit:

• Button states (Normal, Hover, Pressed, Selected, Disabled)
• UI panels and window frames
• List item components
• Progress bars, sliders, and scrollbars
• Light and dark icon variations
• Decorative lines, accents, and detail elements
• Controller UI icons
• Circle Menu elements
• Dark background textures
• Preview images for easy reference

All files are provided in PNG format and scale cleanly across different resolutions.

====
How to Use

Import the PNG files into your preferred game engine.

Place panels, buttons, icons, or decorative elements onto your Canvas or Widget Blueprint.

Combine elements freely to build main menus, option screens, HUDs, inventory layouts, and more.

If you want to match your project’s color palette, feel free to apply tinting or blend modes such as Multiply or Overlay.

If you are using Unreal or Unity, you may also turn frequently used elements into reusable components (Widgets or Prefabs) to speed up your workflow.

====
Customization Tips

• Light color adjustments with tinting work very well for this kit.
• Keeping consistent spacing and padding will help maintain a clean overall look.
• Button state images are aligned to make switching between states as simple as possible.
• You can choose between light or dark icons depending on the background tone of your game.

Feel free to experiment. The assets were made to be flexible and adapt easily to many styles.
